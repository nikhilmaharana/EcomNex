A modern full-stack eCommerce web application built using the MERN stack with clean UI, modular architecture, and scalable backend.

---

## 🚀 Features
- 🔐 JWT Authentication
- 🛍️ Product Listing & Details
- 🖼️ Image Gallery
- 🛒 Cart System
- ⭐ Reviews & Ratings
- 🔍 Search & Filter
- 📱 Responsive Design

---

## 🧱 Tech Stack
Frontend: React (Vite), JavaScript, CSS  
Backend: Node.js, Express.js  
Database: MongoDB (Mongoose)  
Auth: JWT

---

## ⚙️ Setup
git clone https://github.com/nikhilmaharana/EcomNex.git  
cd EcomNex  

cd client && npm install  
cd ../server && npm install  

---

## ▶️ Run
Backend:
cd server  
npm run dev  

Frontend:
cd client  
npm run dev  

---

## 🌐 API
GET /api/products  
GET /api/products/:id  
POST /api/auth/register  
POST /api/auth/login
